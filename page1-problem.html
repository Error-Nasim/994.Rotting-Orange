üçä Understanding the Rotting Oranges Problem

Let's break down EXACTLY what we're trying to solve

The Story: A Warehouse Full of Oranges

Imagine you work at a fruit warehouse. Your oranges are stored in a rectangular grid (like a shelf with rows and columns). Every cell in this grid can be:

0
Empty Cell

No orange here. Maybe it's empty space or a blocked section.

1
Fresh Orange

A perfectly good, healthy orange ready to sell.

2
Rotten Orange

This orange has gone bad and will spread rot to its neighbors.

The Problem: Rot Spreads Like Disease

Here's the critical rule that drives this entire problem:

üö® Every minute, a rotten orange will infect all FRESH oranges directly adjacent to it.

"Adjacent" means up, down, left, or right (NOT diagonally).

Let's See It Happen

Imagine this starting grid:

Minute 0 (Initial State)

2 1 1
1 1 0
0 1 1

We start with 1 rotten orange at position (0,0) and 7 fresh oranges scattered around.

Minute 1

2 2 1
2 1 0
0 1 1

The original rotten orange (0,0) infects its neighbors: right (0,1) and down (1,0). They become rotten NOW.

Minute 2

2 2 2
2 2 0
0 1 1

Now the oranges that became rotten in minute 1 spread to THEIR neighbors. The rot is expanding like a wave!

Minute 3

2 2 2
2 2 0
0 2 1

The wave continues spreading. Notice (2,1) just got infected from (1,1).

Minute 4 (Final)

2 2 2
2 2 0
0 2 2

‚úÖ ALL fresh oranges are now rotten! The answer is 4 minutes.

What Are We Asked to Find?

Question: Given a grid, return the MINIMUM number of minutes until NO fresh oranges remain.

Special Case: If it's IMPOSSIBLE for all oranges to rot (e.g., a fresh orange is completely isolated), return -1.

Example Inputs & Outputs

Example 1: Normal Case

Input:

grid = [[2,1,1],
        [1,1,0],
        [0,1,1]]

Output: 4

Why? As we saw above, it takes 4 minutes for all oranges to rot.

Example 2: Isolated Orange

Input:

grid = [[2,1,1],
        [0,1,1],
        [1,0,1]]

Output: -1

Why? Look at position (2,0) - that fresh orange is surrounded by empty cells and can NEVER be reached by rot!

0 1 1
1 0 1

The orange in red is trapped!

Example 3: No Fresh Oranges

Input:

grid = [[2,2],
        [2,2]]

Output: 0

Why? All oranges are already rotten! It takes 0 minutes.

Key Insights Before Moving On

‚úÖ Rot spreads simultaneously from all rotten oranges at once (not one at a time)

‚úÖ Rot spreads in 4 directions only: up, down, left, right

‚úÖ We need to track TIME (minutes) as the spread happens

‚úÖ Some oranges might be unreachable ‚Üí return -1
