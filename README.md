### Problem Overview

Given a grid representing a warehouse of oranges where:

- `0` = Empty cell
- `1` = Fresh orange
- `2` = Rotten orange

Every minute, rotten oranges infect adjacent fresh oranges (up, down, left, right). The goal is to determine the minimum time for all oranges to rot, or return `-1` if impossible.
